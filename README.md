# Football Dynasty

Mobile-first incremental football career sim prototype.

## Run Locally

```bash
npm install
npm run dev
```

## Build

```bash
npm run build
```

## Current Prototype

- React + Vite + TypeScript.
- Mobile-first Player dashboard.
- Bottom nav with Player, Training, Club, Home.
- Large central advance-time button.
- Dark/lime visual direction from `DESIGN.md`.
- Functional early game state.
- Training focus and intensity affect XP, fitness and trust.
- Match Day runs live with an advancing clock, simulated team/opponent events, and paused player moments.
- Live match supports 1x/2x/4x speed and skip controls for next event, highlight, or sim full time.
